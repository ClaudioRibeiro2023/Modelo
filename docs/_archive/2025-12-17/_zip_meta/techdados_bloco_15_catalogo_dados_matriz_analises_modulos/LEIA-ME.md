# TechDados — Bloco 15: Catálogo de Dados + Matriz de Análises + Árvore de Módulos (MVP)

Data: 2025-12-17

## Objetivo

Transformar toda a documentação de análise e bases de dados (que você enviou) em **documentos canônicos do TechDados**,
prontos para guiar o desenvolvimento do produto e das telas.

Este bloco entrega:

- **Catálogo de datasets** (o que existe, granularidade, chaves, refresh, qualidade)
- **Dicionário de dimensões e métricas**
- **Modelo dimensional sugerido** (star schema) para BI
- **Matriz de análises (MVP + expandido)** com:
  - objetivo, KPI, dataset(s) necessários, filtros, visualizações, export, RBAC sugerido
- **Árvore de módulos/submódulos** do produto (sidebar)
- **Template de novas análises** (padrão TechDados)

## Como aplicar

1. Extraia o ZIP na raiz da repo.
2. Siga o guia canônico:

- `docs/00-auditoria/_blocos/2025-12-17_bloco_15_catalogo_dados_matriz_analises_modulos.md`

## Observação

Este bloco NÃO substitui o PDF de hierarquia; ele cria a **matriz operacional** do produto e referencia o PDF como fonte normativa.
